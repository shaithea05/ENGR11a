Name Tag

Project 1 - Basic Print

The goal of this assignment was to create my first personalized 3D print design using a public 3D design and PrusaSlicer. This project allowed me to be exposed to the PRUSA Mini machine and learn how to customize public designs through different texts and fonts. If you'd like to read through further details about the steps, feel free to click this link to the Prusa MINI+ Handbook

I started this process by downloading a standard name tag file, and uploading it to Prusa Slicer, where I directly edited the text to display my name. Once I completed designing it, I sliced the project in order to see the process of the print at any point, and exported the G-code to transfer to the machine. As this was my first project I did not change any of the settings, but I did look through the different options that were available to me. It was a little difficult as it was all new to me; but it was very interesting to explore the software and see all of the different changes I could make along with how they would affect the final print. It is important to note that at this point I had to make sure that I was paying attention to the types of files I was working with as there were multiple moving parts. I had to make sure that my STL file became a 3MF file as it would then carry the complete model information required for printing.

At the machine, I attempted to load the filament for the base of the name tag but initially struggled with it as I had thought I inserted the filament far enough into the tube, when I really had not. After talking with some others, I tried again and made sure to keep pushing the filament until I felt the machine begin to tug at it. Once I felt that initial pull, I could let the machine take over. After loading the filament into the 3D printer properly, I connected the USB to upload the data that I had transferred from PrusaSlicer, and finally began printing. Partway through the printing—once the base was fully printed—I unloaded the filament by picking the associated settings on the machine, and swapped it with a different colored filament for the text, design, and border. Once the printing was fully completed, I removed the magnetic plate, and flexed the build plate in order to pop my name tag out from the machine.

After completing this project, I noticed that although the front side was relatively clean minus the stringing along the name, the back of my name tag was filled with gaps. The stringing could be prevented by possibly changing up the temperature or the speed of the retraction or the movement. The under extrusion may have been due to an incorrect calibration of the filament diameter or the extrusion multiplier being too low. Additionally, there was some curling on the sides along with some visible lines on the sides of the print which could be due to some overheating issues that may have caused the filament to change shape over time as it did not have enough time to cool down. Overall, this print was pretty interesting to me as I had the chance to customize the design with the text, font, and font size. I am excited to work on more projects where I am able to design aspects of the print by hand!

Ring Holder

Project 2 - Calibration Print

The goal of this project was to gauge the limitations of what is possible to design for 3D printing with the 3D printers here. I decided to focus specifically on whether the printer would be able to handle printing overhangs at many different angles with this ring bowl.

Once I downloaded the public design, I imported it into PrusaSlicer. The main change I made to this file were the size of the print—I enlarged it to create more difficulties in the print. After editing the design in PrusaSlicer, I sliced the STL file to export the gCode, saved the project as a 3MF file, and then prepared to transfer the design to the machine. The image above shows the design in PrusaSlicer after I sliced it, and as you can see, there are many sections that had potential to be subject to overhang issues while printing. It is important to differentiate between STL files and 3MF files as STL files primary focus on the geometry of the object and only contains the basic mesh that is printed, whereas the 3MF file is made specifically for 3D printing, and has all of the model information (in addition to the mesh), allowing more ease in using other softwares.

The image above was taken part way through the print. At this stage, the print was going well and there seemed to be no issues. I was slightly disappointed as I wanted there to be some challenges here; however, there were no crazy angles yet where the machine would have trouble with any sort of overhang. Once the print was completed, I repeated the same steps of removal as the name tag—I removed the magnetic plate and flexed the build plate in order to pop the ring holder off of the plate.

Once the print was completed, it could be seen that there was under extrusion as there were many gaps at the base of the object which may have been due to the filament diameter not being calibrated properly between the machine and the filament or the extrusion multiplier being too low. There was also some stringing between the vines which could have been because of some overheating. And finally, there was some overhang along the sides as I had hoped for, which was a challenge the machine was unable to fully overcome. Although this print ended up looking pretty good, this design was a successful fail as it exemplified a few different issues with the final print!

Reflection #1

Reflection on my first 3 projects.

This was my first time using a 3D printer since middle school and I really enjoyed it! There were a lot of things that I did not know, but I loved trying everything out for the first time and being able to take the time to explore the different features of the software. I had never loaded and unloaded filament to a 3D printer before, and it was MUCH more difficult than it may have ever looked. Additionally, there were a lot of terms that I had not previously known that I got a chance to learn such as STL, 3MF, etc. I feel as though within a short time I was able to learn and try out a lot of different things, and I am excited to learn more as the semester goes on!
